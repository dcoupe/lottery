#javascript抽奖程序
=======

*JavaScript+HTML+CSS开发而成, 支持IE6+,Chrome,Firefox等浏览器。
*抽奖过程全键盘操作, 只需要按回车键即可。
*支持活动标题, 奖项, 候选人, 摇奖时的跳转速度等设置。
